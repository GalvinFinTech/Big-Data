
Em chào thầy,

Em tên Nguyễn Hoàng Vi, nhóm trưởng nhóm 2 xin phép nộp bài ạ!

Dự án của nhóm em làm liên quan đến phân cụm khách hàng và tạo ra hệ thống gợi ý sản phẩm được xây dựng để phân tích dữ liệu khách hàng và đưa ra các gợi ý sản phẩm phù hợp. 
Hệ thống sử dụng thuật toán Collaborative Filtering để phân tích hành vi của khách hàng.

Để xem chi tiết và tải xuống tài liệu, thầy vui lòng truy cập thư mục Google Drive tại đây:
[https://drive.google.com/drive/folders/1VKl13o2U92kdMG5rMDanLBIBQfgyPxOa?usp=share_link]

Cấu trúc dự án:

- src: Chứa mã nguồn Python của hệ thống.

    + main.py: File chạy chính của hệ thống.
    + rcm_sys.py: File chứa code cho ứng dụng web.
- data: Chứa dữ liệu đầu vào, bao gồm file 'OnlineRetail.csv.'

- docs: Chứa tài liệu và tài nguyên.

    + '241CN1302_Nhóm2.pdf': Báo cáo chi tiết quá trình xây dựng hệ thống.
    + 'Wed_app.mov': Video demo giới thiệu ứng dụng web.

Cảm ơn thầy đã tận tình dạy dỗ chúng em thời gian qua, chúng em chân thành cảm ơn thầy và chúc thầy thật nhiều sức khoẻ, niềm vui và may mắn ạ!
